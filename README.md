## Формат защиты лабораторных работ

- Каждое задание демонстрируется в формате собеседовании:
    - запуск локальной версии приложения;
    - демонстрация работы ключевых функций;
    - объяснение архитектуры и принятых решений;
    - ответ на технические вопросы по коду.
